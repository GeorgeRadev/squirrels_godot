# squirrels_godot
Simple GODOT 2 players game in Squirrels Theme collecting acorns

## how to use 
- Import project in godot
- Set control keys in: Project -> Project Settings -> Input Map
- Start the project (usualy **F5**)
- Collect at least 10 acorns each
